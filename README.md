# 📚 BookAbode – Client

BookAbode is a modern full-stack online book marketplace where users can discover, explore, wishlist, and manage books through a clean and responsive interface. The frontend is built with **Next.js**, **TypeScript**, and **Tailwind CSS**, focusing on performance, accessibility, and an engaging user experience.

> **Status:** Active Development 🚀

---

## 🌐 Live Website

**Client:** https://book-abode.vercel.app

**Server API:** https://book-abode-server.onrender.com

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Safa-Anan08/Book-Abode
```

Go to project

```bash
cd bookabode-client
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env.local` file.

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api

```

---

# 📦 Build

```bash
npm run build
```
